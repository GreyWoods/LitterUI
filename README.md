## 开始使用

1.添加CSS

需要在CSS中开启border-box

```
*,*::before,*::after{box-sizing: border-box;}
```

2.安装gulu
```
npm i --save xcsf-vue
```

3.引入gulu
```
import {Button, ButtonGroup, Icon} from 'xcsf-vue'
```
